# Code of Conduct

## Our Standard

CapTest is a private, proprietary project. All individuals with any level of access to this repository — whether as contributors, reviewers, or read-only viewers — are expected to conduct themselves professionally, responsibly, and with integrity.

---

## Expected Behavior

- Communicate respectfully and professionally at all times.
- Treat all repository content, API details, credentials, and access information as strictly confidential.
- Use your access solely for its authorized, intended purpose.
- Report any security concerns, policy violations, or misuse through the appropriate channels.
- Respect the intellectual property and proprietary nature of this project.

---

## Unacceptable Behavior

- Sharing, distributing, or exposing any source code, API endpoints, authentication tokens, or service configurations without explicit authorization.
- Cloning, forking, downloading, or using any part of this codebase or its connected APIs without permission.
- Accessing or probing backend services or APIs discovered through this frontend code.
- Harassment, discrimination, or hostile communication of any kind.
- Misrepresenting your identity or level of authorization to gain access.
- Using any part of this repository or its connected services for personal, commercial, or external purposes.

---

## Enforcement

Violations of this Code of Conduct may result in:

- Immediate and permanent revocation of repository access.
- Escalation to organizational or legal authorities, depending on the severity of the violation.
- Civil or criminal legal action, where applicable.
